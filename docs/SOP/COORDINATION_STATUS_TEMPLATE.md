# Coordination status template

Use this block whenever one agent reviews, continues, or depends on another agent's work.

```text
COORDINATION STATUS
Agreement: aligned | partial | conflict | unknown
Compared: <charter / issue / PR / commit / files>
Disagreement: <none or concise statement>
Evidence gap: <none or missing proof>
Ownership overlap: <none or overlapping paths/state>
Risk if unresolved: <none or consequence>
Recommended default: <one action>
Founder decision required: yes | no
```

`Aligned` requires comparison against an identifiable GitHub artifact or current evidence. Do not silently resolve canon, priority, definition-of-done, or ownership conflicts.
