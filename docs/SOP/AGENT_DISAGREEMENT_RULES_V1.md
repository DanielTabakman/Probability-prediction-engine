# Agent disagreement rules v1

This file is a short alias for the mandatory reconciliation rules in [`CHATGPT_GITHUB_CODEX_CONTROL_PLANE_V1.md`](CHATGPT_GITHUB_CODEX_CONTROL_PLANE_V1.md).

Agents must surface, not conceal:

- canon conflicts
- implementation conflicts
- missing or contradictory evidence
- overlapping ownership
- priority drift

Use [`COORDINATION_STATUS_TEMPLATE.md`](COORDINATION_STATUS_TEMPLATE.md). Product and canon disagreements require founder resolution; technical disagreements default to fresh evidence and the smallest reversible implementation.
