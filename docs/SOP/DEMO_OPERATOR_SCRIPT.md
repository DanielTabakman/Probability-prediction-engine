# Demo operator script (~5 minutes)

For **public demo** (`PPE_ENABLE_SNAPSHOTS=0`) or live walkthrough. Exploration only — not advice.

## Setup (30s)

- URL: `https://marketstructureos.com` (demo) or local with demo env:
  - `PPE_ENABLE_SNAPSHOTS=0`
  - `PPE_PRIVATE_APP_URL=https://app.marketstructureos.com` (optional CTA)
- Tab title: **Probability Engine**

## Script

1. **Intro (30s):** “This is a BTC options research cockpit. Purple/orange curves are market-implied vs a reference benchmark — not a trade recommendation.”

2. **Tutorial (30s):** Open **How to use this demo (~2 min)**; collapse with **Got it** when done.

3. **Sidebar (45s):** Keep **Bitcoin view** on; click **Refresh priced inputs (Deribit)** once; pick an **Expiry**.

4. **Implied lab (90s):** Scroll to **Bitcoin implied lab**. Point at the chart (market-implied anchor). In MVP1 compact mode, click a **preset** — note the green payoff line moves. Read **MVP1 output** (watch only / candidate / no trade).

5. **Disagreement (45s):** Open **Belief vs market — at a glance** (or **Review & disagreement digest**). Explain width or location tension in plain language.

6. **Full app CTA (30s):** On demo, click **Get full access** → full app saves freezes and reviews.

7. **Close (30s):** “We freeze what the system saw before outcomes so cases can be reviewed later — that’s on the private app, not this public demo.”

8. **Log session (60s):** Open **Learn** (`/learn`) or finish the Strategy Lab tour — use **Log this session**. Copy the validation row or run `log_demo_session.cmd` on your desktop repo after the call.

## Reality Check #1 (demo clarity)

After the session, ask: “What was the main object on screen?” Pass if they mention the chart / market-implied curve / disagreement readout.
