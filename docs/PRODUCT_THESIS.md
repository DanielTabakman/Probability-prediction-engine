# Product Thesis

## North star
A visual lab that translates market-implied belief, user belief, and option payoff geometry into each other in real time.

## Current milestone
Build a trustworthy manual options playground where users can:
- see the market-implied distribution
- input their own belief
- manually adjust option structures
- see payoff and tradeoffs update in real time
- inspect calculations for trust and intuition-building

## In scope for current sprint
- cleaner one-screen layout
- chart visible near the top
- controls on the left
- chart and summary on the right
- advanced calculations collapsed by default
- clean mode toggle for:
  - exact strikes
  - target payoff
- clean belief view toggle for:
  - market belief
  - user belief
  - compare

## Out of scope for current sprint
- AI assistant
- prediction market integration
- automatic trade recommendation engine
- full market coherence scanner
- frontend stack migration
