# Contributing

Keep contributions small and easy to review.

## Setup

Use the repo’s standard workflow:

- Create a venv (`python -m venv .venv`)
- Install deps (`pip install -r requirements.txt`)

## Tests

Pytest is used for tests.

```bash
python -m pip install pytest
python -m pytest
```

## Formatting

No additional formatting/lint toolchain is required for contributions right now. Prefer minimal, readable diffs that match surrounding code style.
